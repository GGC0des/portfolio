Web App PetPatron

My idea for PetPatron - a web app to locally connect animal lovers with shelters - was selected as a final project at the LeWagon Bootcamp. Chosen as Team Lead by my peers and group members I managed Trello workflows (Kanban method), lead daily stand-ups and reviewed pull requests on GitHub. I programmed 80% of the backend in Ruby on Rails, including dashboard logic for different types of users & multi-search including tags & Mapbox. On the frontend I contributed JavaScript features using Stimulus framework for search elements and different views depending whether the user is logged in or not.
